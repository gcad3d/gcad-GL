=================== Functions app:
- key '1' or '2' or '0' - select new rotation-centerpoint


  Create-objects:
- key 'p' or button "Point" to add point
- key 'l' or button "Curve" to add polygon-curve
- key 't' or button "Surface" to add triangle-surface
 

  Attributes:
- key 'c' or button "change-Color" to change color for following objects
- key 'h' to hide or reDisplay obj (last -1)
- key 'w' to increase width of point, curve for following objects

- key 'd' to dump internal data
- key 'Esc' to exit.
